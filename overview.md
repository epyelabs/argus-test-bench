# Installation

1. Setup CSI/MIPI Camera
2. Setup UVC Camera
3. Setup SIM7600G
4. IMU
   - Ask which pin connected
   - Default to GPIO2: SDA, GPIO3: SCL
5. MIC
6. RGB LED

# Interface

## Camera

- Ability to take a picture (by camera source)
- Ability to record video (by camera source)

## 4G/LTE Module

- Ability to read the CSQ
- Ability to read GSP Data

## IMU

- Ability to read the data

## MIC

- Ability record audio (with VU meter)

## RGB LED

- Ability to turn on/off the lights

# Pre-reqs

1. github
2. minicom
3. nodejs
