# Installation

1. Setup CSI/MIPI Camera
2. Setup UVC Camera
3. Setup SIM7600G
4. IMU
   - Ask which pin connected
   - Default to GPIO2: SDA, GPIO3: SCL
5. MIC
6. RGB LED

# Test bench

## Camera

### Per camera source

- Ability to take a picture (or pictures)
- Ability to record video (with additional settings like fps, resolution, etc)

## 4G/LTE Module

- Ability to read the CSQ
- Ability to read GSP Data

## IMU

- Ability to read the data

## MIC

- Ability record audio (with VU meter)

## RGB LED

- Ability to turn on/off the lights
